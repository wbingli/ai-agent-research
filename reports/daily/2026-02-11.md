### 3. üî¨ GLM-OCR ‚Äî 0.9B Parameter OCR Model, #1 on OmniDocBench
- **Link:** https://github.com/zai-org/GLM-OCR
- **What:** Tiny (0.9B params) multimodal OCR model from Zhipu AI that tops OmniDocBench V1.5 at 94.62. Handles formulas, tables, seals, complex layouts.

- **Why interesting:** State-of-the-art OCR at a fraction of the size. Supports vLLM, SGLang, Ollama deployment. Open weights on HuggingFace. Perfect for RAG pipelines that need to ingest documents. From same team that just released GLM5.

### 5. üéÆ AI Ported SimCity (1989) to Browser in 4 Days
- **Link:** https://garryslist.org/posts/ai-just-ported-simcity-in-4-days-without-reading-the-code
- **What:** Christopher Ehrlich pointed OpenAI's codex-5.3 at the entire SimCity C codebase (assembly-ported-to-C from Commodore 64 era) and got a working browser port in 4 days. No human code reading.
- **How:** Key technique: wrote a bridge to call original C code, then ran property-based tests asserting TypeScript port matched C behavior. Agent iterates until tests pass. Spec + verification, not manual understanding.
- **Why interesting:** Demonstrates a new pattern ‚Äî "specification and verification" coding where the human never reads the legacy code. Every COBOL banking system, every ancient government codebase is now potentially portable. $200/month ChatGPT subscription, no rate limits hit.
