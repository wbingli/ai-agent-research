### 2. Anthropic Cowork — Claude Desktop Agent for Non-Coders
**Link:** Claude Desktop (macOS)
**What:** Anthropic launched Cowork, a new mode in Claude Desktop that gives non-technical users agent capabilities — file management, document processing, organization — without any coding. Born from observing that many Claude Code users were using it for knowledge work, not code. Now has a "Projects" feature for persistent workspace context across sessions.
**Why interesting:** This is Anthropic's play to expand the agent surface area beyond developers. The insight that people use coding agents for non-coding tasks is spot-on (we do it ourselves!). The persistent project/workspace model addresses context drift — a real pain point for research-heavy work. Worth watching how this evolves.

### 3. oh-my-openagent — Multi-Agent Orchestration Plugin for OpenCode
**Link:** https://github.com/code-yeongyu/oh-my-openagent (v3.12.0)
**What:** A plugin that turns a single LLM into a development team lead. Uses a "Sisyphus" orchestrator that delegates to specialized agents by category. 5-stage initialization pipeline, 8 OpenCode hook handlers, and 4-step model resolution. Supports multiple LLM providers (Claude, GPT, Gemini) with auto-switching prompts per model family. Background agent queue with concurrency management.
**Why interesting:** The architecture is well-thought-out — category-based agent delegation, provider-aware prompt switching, and a persistent loop that continues until 100 iterations or user interrupt. It's a serious take on "one prompt, many specialists" that doesn't just split by role but adapts to the underlying model.

### 4. learn-claude-code — Bash-Only Agent Framework
**Link:** https://github.com/shareAI-lab/learn-claude-code (GitHub Trending)
**What:** A nano-scale agent framework that replicates Claude Code's core functionality using only Bash scripts. No Python, no Node, no dependencies. Trending on GitHub as an educational project showing how agentic coding assistants work under the hood.
**Why interesting:** Pure educational value. Strips away all the framework complexity to show the core loop: read context → call LLM → parse tool calls → execute → loop. If you want to understand how Claude Code actually works at a conceptual level, this is a weekend read.
