# Tech Digest — 2026-07-08

**1. GPT‑Live**
OpenAI introduced a new generation of voice models for more natural real-time human-AI interaction, now powering ChatGPT Voice. For engineering leaders, this pushes voice from a demo feature toward a production interface pattern for support, assistants, and multimodal workflows. [Read](https://openai.com/index/introducing-gpt-live/)

**2. TypeScript 7**
Microsoft announced TypeScript 7.0, a native port focused on dramatically faster compiler and tooling performance. If the claimed speedups hold in large repos, this is a meaningful productivity and CI-cost improvement for frontend and full-stack teams. [Read](https://devblogs.microsoft.com/typescript/announcing-typescript-7-0/)

**3. Cloudflare Meerkat: globally distributed consensus**
Cloudflare Research described Meerkat, an experiment in global consensus using a new algorithm called QuePaxa to build strongly consistent, fault-tolerant services across regions. It is worth tracking because it targets one of the hardest infrastructure tradeoffs: low-latency global systems without giving up correctness. [Read](https://blog.cloudflare.com/meerkat-introduction/)

**4. GitLost: tricking GitHub’s AI agent into leaking private repos**
Noma Security disclosed a prompt-injection vulnerability in GitHub agentic workflows that could let a crafted public issue exfiltrate data from private repositories in the same organization. This is a concrete reminder that AI coding agents need the same threat modeling as any privileged automation surface. [Read](https://noma.security/blog/gitlost-how-we-tricked-githubs-ai-agent-into-leaking-private-repos/)

**5. Rewriting Bun in Rust**
The Bun team explained why and how they rewrote major parts of Bun from Zig to Rust. Beyond language preference, the interesting takeaway is organizational: runtime teams are optimizing for maintainability, ecosystem leverage, and contributor velocity as much as raw performance. [Read](https://bun.com/blog/bun-in-rust)
