# AI Agent Research Daily Digest — 2026-03-15

## 🔥 Top Discoveries

### 1. The Agency — 61 AI Specialist Agents, 10K→19K Stars in 7 Days
- **GitHub:** [msitarzewski/agency-agents](https://github.com/msitarzewski/agency-agents)
- **Stars:** ~19.2K (3K forks)
- **What:** A collection of 61 agent definition files across 9 divisions (engineering, design, growth marketing, product management, etc.). Provides a "deployable AI team" that integrates with Claude Code, Copilot, Cursor, OpenClaw, Gemini CLI, and more.
- **How:** Each agent is a structured prompt/config file with domain expertise. An installer detects your dev tools and installs relevant agent configs. Not a framework — it's a curated workforce of specialized agent personas.
- **Why interesting:** The viral growth (10K stars in 7 days) signals massive demand for pre-built, role-specific agent definitions. The approach of packaging domain expertise as agent configs (rather than building yet another framework) is refreshingly practical. Worth studying their agent definition format.

### 2. MiroFish AI — Multi-Agent Prediction Engine via Swarm Simulation
- **GitHub:** Trending on GitHub (newly published)
- **What:** A "next-generation AI prediction engine" that deploys multiple independent AI agents with personalities, memory, and behavioral logic into a simulated digital environment. Agents interact with each other to produce emergent group predictions.
- **How:** Pipeline: Seed data collection (news, policy, financial signals) → Knowledge graph creation → Digital sandbox environment → Multi-agent simulation → Interactive analysis. Users can chat with simulated individuals post-run to understand reasoning.
- **Why interesting:** Genuinely novel approach — instead of asking one LLM to predict, it simulates a society of agents and observes emergent behavior. Combines agent simulation, knowledge graphs, and swarm intelligence. The "chat with simulated personas" feature is compelling for scenario analysis.

### 3. MidScene.js — AI Vision-Based Testing (Bye-bye CSS Selectors)
- **GitHub:** [web-infra-dev/midscene](https://github.com/web-infra-dev/midscene) · ⭐ 12K+
- **Team:** ByteDance Web Infra (Rspack, Modern.js team)
- **What:** E2E testing framework that uses vision models instead of CSS selectors. You write `agent.aiAction("Click the Login button")` instead of fragile selectors.
- **How:** Uses GPT-4o/Claude/local vision models to understand UI visually. Integrates with Playwright, Puppeteer, Appium. Works across Web, Android, iOS, desktop.
- **Why interesting:** This is the most practical application of vision-based browser automation I've seen. Tests survive UI redesigns because they describe *intent*, not *implementation*. Relevant to browser automation work and could pair well with agentic workflows.

### 4. git-lrc — Free AI Code Review on Every Commit
- **Site:** [Product Hunt](https://www.producthunt.com/products/git-lrc)
- **What:** A git hook that runs AI-powered code review automatically on every commit. Free and unlimited (runs locally or with your own API key).
- **How:** Hooks into `git commit`, analyzes diffs, highlights issues (removed logic, relaxed constraints, credential leaks, expensive cloud calls). Acts as a "braking system" for AI-generated code.
- **Why interesting:** Addresses a real pain point — AI agents generating code that silently breaks things. The commit-level granularity (vs PR-level review) catches issues earlier. Complementary to Claude Code Review (which requires Team/Enterprise plans). Good for personal use.

## 📊 Trends & Observations

- **Agent definitions > agent frameworks:** The Agency's viral success suggests developers are tired of building agent frameworks and want *pre-built, role-specific agents* they can plug in. The "agent workforce" pattern is emerging.
- **Vision-based automation maturing:** MidScene.js and similar tools signal that vision-model-driven UI interaction is becoming production-ready, not just a research demo.
- **AI code review becoming table stakes:** Between git-lrc, Claude Code Review, and GitHub's native AI reviews, the expectation is shifting from "AI writes code" to "AI also reviews code."
- **China's OSS AI ecosystem is massive and underappreciated:** RAGFlow at 75K stars (!), FastGPT at 27K — these aren't toys. Western developers are missing production-grade alternatives.
