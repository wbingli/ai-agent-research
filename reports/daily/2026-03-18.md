### 3. 🔐 GitHub MCP Server Gets Secret Scanning
**Source:** [GitHub Changelog (Mar 17)](https://github.blog/changelog/2026-03-17-secret-scanning-in-ai-coding-agents-via-the-github-mcp-server/)

GitHub's MCP Server can now scan code changes for exposed secrets *before* you commit or open a PR. This is directly relevant given GitGuardian's report showing **24,008 unique secrets exposed in MCP config files** on public GitHub, an 81% surge in AI-service credential leaks year-over-year.

**Why it matters:** As AI coding agents commit more code autonomously, secret leakage is a growing attack surface. This is the first major guardrail integrated directly into the MCP workflow.

### 5. 🗺️ MCP 2026 Roadmap Published
**Source:** [MCP Blog (Mar 9)](http://blog.modelcontextprotocol.io/posts/2026-mcp-roadmap/)

Lead maintainer David Soria Parra outlined four focus areas:
1. **Streamable HTTP transport** — horizontal scaling for production deployments
2. **Tasks primitive** — closing lifecycle gaps
3. **Enterprise readiness** — audit trails, SSO

The spec hasn't changed since Nov 2025, but production pain points are driving the roadmap. SurePath AI launched MCP Policy Controls (Mar 12) for governance.
