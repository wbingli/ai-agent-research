# Tech Digest — 2026-06-11

**1. EC2’s Formally Verified Isolation Engine**
AWS describes how it split a small “separation kernel” out of Nitro’s security system and implemented it in a restricted Rust subset so the VM-isolation boundary could be formally verified. For engineering leaders, this is a useful signal that cloud security is moving from best-effort hardening toward mathematically checked infrastructure primitives. [Read](https://www.amazon.science/blog/ec2s-formally-verified-isolation-engine-provides-mathematical-assurance-of-virtual-machine-isolation)

**2. How Discord Moved Voice to the Edge**
Discord explains the migration of voice and video traffic onto Cloudflare’s edge network, including the latency wins and real operational bugs encountered along the way. It is a practical case study in media-serving architecture: moving compute closer to users can improve experience, but only if observability, routing, and failure handling are ready. [Read](https://discord.com/blog/how-we-moved-discord-voice-to-the-edge)

**3. The RCE That AMD Wouldn’t Fix**
A researcher details a man-in-the-middle remote-code-execution flaw in AMD Ryzen Master and the disclosure friction before AMD shipped a fix. The lesson is less about one vendor and more about supply-chain trust: privileged desktop utilities and update paths deserve the same threat modeling as server agents. [Read](https://mrbruh.com/amd2/)

**4. Open Reproduction of DeepSeek-R1**
Hugging Face’s Open-R1 project continues the push toward fully open reproductions of frontier-style reasoning models, with code and training artifacts intended to make the recipe inspectable. This matters because model capability is becoming an engineering dependency, and reproducible training pipelines reduce vendor lock-in and black-box risk. [Read](https://github.com/huggingface/open-r1)

**5. Homebrew 6.0.0**
Homebrew 6.0.0 adds a tap-trust security mechanism, a faster and smaller internal JSON API, Linux sandboxing, performance work, and early macOS 27 support. For teams that rely on developer workstations as part of the production toolchain, package-manager security and reliability are real engineering infrastructure, not just local convenience. [Read](https://brew.sh/2026/06/11/homebrew-6.0.0/)
