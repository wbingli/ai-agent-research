### 2. Computer use is 45x more expensive than structured APIs

- **Why it matters:** A timely reality check for agent architecture. The piece argues that UI-driven “computer use” agents are dramatically more expensive than structured API calls, reinforcing a practical design principle: give agents typed tools and stable interfaces wherever possible, and reserve visual/browser automation for gaps in the system boundary.
- **Source:** Hacker News
- **URL:** https://reflex.dev/blog/computer-use-is-45x-more-expensive-than-structured-apis/
- **HN discussion:** https://news.ycombinator.com/item?id=48024859

### 3. Formatting an entire 25 million line codebase overnight: the rubyfmt story

- **Why it matters:** Stripe’s rollout of a Rust-based Ruby formatter across one of the world’s largest Ruby codebases is a strong developer-infrastructure case study. The interesting bits are not just formatter implementation, but migration strategy: safe large-scale codemods, CI/developer workflow integration, performance, and minimizing merge-conflict pain during a whole-repo mechanical change.
- **Source:** TLDR Tech
- **URL:** https://stripe.dev/blog/formatting-an-entire-25-million-line-codebase-overnight-the-rubyfmt-story

### 4. Should I run plain Docker Compose in production in 2026?

- **Why it matters:** A pragmatic backend operations article: Docker Compose can still be viable for small production systems, but only if you explicitly close gaps around health/healing, updates, image pinning, cleanup, socket security, and observability. Useful reading for teams deciding when Kubernetes is overkill versus when orchestration requirements are real.
- **Source:** Hacker News
- **URL:** https://distr.sh/blog/running-docker-in-production/
- **HN discussion:** https://news.ycombinator.com/item?id=47962032

### 5. Security Advisory: Local privilege escalation in Lix and Nix

- **Why it matters:** Nix/Lix daemon buffer-overflow vulnerabilities can allow local attackers to execute arbitrary code as the daemon user, root in common multi-user installs. For engineering teams using Nix in CI, developer machines, or build infrastructure, this is a patch-now item and a reminder that “developer tooling” often sits on highly privileged paths.
- **Source:** Lobste.rs
- **URL:** https://discourse.nixos.org/t/security-advisory-local-privilege-escalation-in-lix-and-nix/77407
- **Lobste.rs discussion:** https://lobste.rs/s/g0crsm/security_advisory_local_privilege
