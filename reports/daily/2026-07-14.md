# Tech Digest — 2026-07-14

**1. Bonsai 27B runs a 27B-class model on a phone**
PrismML announced a Qwen3.6-based multimodal model designed for sustained local agentic workflows, including tool use and MCP integration. If the claims hold up, it pushes meaningful AI inference closer to private, offline, zero-marginal-cost product experiences. [Read](https://prismml.com/news/bonsai-27b)

**2. Cursor 0day shows the risk of agentic developer tooling**
Mindgard describes a Windows issue where opening a repository in Cursor could cause a planted `git.exe` in the project root to execute without prompts. It is a sharp reminder that AI IDEs inherit the full supply-chain and workspace-trust threat model of local developer environments. [Read](https://mindgard.ai/blog/cursor-0day-when-full-disclosure-becomes-the-only-protection-left)

**3. Tailscale discloses SSH argument-handling root-access bug**
Tailscale’s latest security bulletin says insecure username argument handling in Tailscale SSH could permit root access in violation of ACLs on affected Linux clients. Teams using identity-aware infrastructure should treat client upgrades as part of the security boundary, not just endpoint hygiene. [Read](https://tailscale.com/security-bulletins)

**4. Job queues are deceptively tricky**
This walkthrough shows how small differences in queue semantics—ordering, retries, visibility, cancellation, and concurrency—change real workload behavior. It is a useful systems-design refresher for anyone building reliable background processing or distributed task execution. [Read](https://typesanitizer.com/blog/job-queues.html)

**5. Dependabot adds a default package cooldown**
GitHub will now wait at least three days before opening Dependabot version-update PRs for newly released packages. The change is a pragmatic supply-chain defense: it reduces exposure to compromised or hastily yanked releases while cutting update noise for engineering teams. [Read](https://github.blog/changelog/2026-07-14-dependabot-version-updates-introduce-default-package-cooldown/)
