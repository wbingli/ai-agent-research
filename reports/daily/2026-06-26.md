**2. AWS Lambda introduces MicroVMs**
AWS launched Lambda MicroVMs: serverless, VM-isolated sandboxes with no shared kernel, fast launch/resume, full lifecycle control, and state preservation for up to 8 hours. This is a big primitive for agent sandboxes, untrusted code execution, CI workers, and multi-tenant compute without running your own fleet. [Read](https://aws.amazon.com/blogs/aws/run-isolated-sandboxes-with-full-lifecycle-control-aws-lambda-introduces-microvms/)

**4. Linux Foundation launches Akrites for AI-enabled open-source security threats**
The Linux Foundation and industry partners launched Akrites to harden critical open-source infrastructure against AI-assisted attacks. The important part is the shift from reactive vulnerability handling to coordinated, ecosystem-level defense as attackers use AI to scale discovery and exploitation. [Read](https://www.linuxfoundation.org/press/linux-foundation-and-industry-leaders-launch-akrites-to-defend-critical-open-source-software-against-ai-enabled-cyber-threats)
