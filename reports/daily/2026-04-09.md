### 3. 🧠 danielmiessler/Personal_AI_Infrastructure (PAI) v4.0.3
- **Repo:** [danielmiessler/Personal_AI_Infrastructure](https://github.com/danielmiessler/Personal_AI_Infrastructure)
- **What:** A full personal AI infrastructure stack built on Claude Code primitives (hooks, slash commands, MCP servers, context files). Positions itself as a "Digital Assistant" that learns, remembers, and improves over time — not just a chatbot.
- **Key features:** Three-tier AI model (chat → tool-use → learning agent), intelligent routing ("say research this and the right workflow triggers"), self-improving behavior with rating/verification feedback loops

### 4. 💻 OpenWork — Open Source Claude Cowork Alternative
- **Repo:** [different-ai/openwork](https://github.com/different-ai/openwork)
- **What:** Open-source desktop app alternative to Claude Cowork/Codex. Local-first with optional remote sharing. Powered by OpenCode underneath.
- **Key features:** Local-first execution, composable (desktop/WhatsApp/Slack/Telegram connectors), CLI orchestrator mode (`openwork start --workspace /path --approval auto`), ejectable to raw OpenCode
- **Why interesting:** The "composable" angle is smart — same agent backend exposed as desktop app, messaging bot, or server. The Telegram/Slack connectors make it a lightweight alternative to full agent platforms for teams that want coding agents in chat.
