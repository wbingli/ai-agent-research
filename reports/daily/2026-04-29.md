# Daily Tech Digest — 2026-04-29

Curated from Hacker News, Lobste.rs, and Simon Willison's weblog with an emphasis on AI agents, LLM tooling, system design, and ad tech at scale.

## Top 5 Stories

1. **PubMatic debuts AgenticOS — agentic platform for programmatic ads**
   **Source:** Marketing Dive · **Why it matters:** Advertisers set objectives, brand-safety rules and creative constraints in their LLM of choice, then a set of AI agents plan, execute, and optimize campaigns in real time. This is the SSP side of the same wave that put Criteo into the ChatGPT ad pilot in March — agentic buying/selling is moving from slideware to live programmatic infrastructure, and it's exactly the surface area to track for ad tech.
   **URL:** https://www.marketingdive.com/news/pubmatic-debuts-agentic-platform-to-solve-programmatics-ai-headaches/808709/
   **Companion read (Criteo + ChatGPT):** https://digiday.com/media-buying/how-criteo-is-turning-llms-into-its-next-big-advertising-channel/

2. **Cloudflare Agents Week 2026 — building "Cloud 2.0, the agentic cloud"**
   **Source:** Cloudflare blog · **Why it matters:** A week of launches aimed at agents as a primary workload: Sandboxes GA (a real shell+filesystem per agent that resumes on demand), Agent Memory as a managed service, Durable Object Facets giving each AI-generated app its own SQLite, an Email Service for agent-native messaging, and AI Gateway as a unified inference layer across 14+ providers. Useful reference architecture if you're designing long-running agents that need state, isolation, and multi-model routing.
   **URL:** https://blog.cloudflare.com/agents-week-in-review/

3. **Simon Willison — "Headless everything for personal AI"**
   **Source:** simonwillison.net · **Why it matters:** Riffs on Matt Webb's thesis that headless services will proliferate because using a personal AI is a better UX than using SaaS UIs directly — and headless services serve agents better. Salesforce already exposes its full platform as APIs/MCP/CLI; if this generalizes, per-seat SaaS pricing breaks. Important framing for anyone planning around MCP, agent-first product surfaces, or the long-term impact on ad-platform UIs.
   **URL:** https://simonwillison.net/2026/Apr/19/headless-everything/

4. **"The Claude Coding Vibes Are Getting Worse"**
   **Source:** Lobste.rs · **Why it matters:** Active thread analyzing recent Claude Code regressions. Diagnosis: adaptive thinking under-allocates reasoning on certain turns — the turns that fabricated had zero reasoning emitted, the deep-reasoning turns were correct. Workaround surfaced in the discussion: `CLAUDE_CODE_DISABLE_ADAPTIVE_THINKING=1` forces a fixed reasoning budget. Worth knowing if you're a heavy Claude Code user.
   **URL:** https://lobste.rs/s/xaicu8/claude_coding_vibes_are_getting_worse

5. **"I Left Port 22 Open on the Internet for 54 Days"**
   **Source:** Lobste.rs · **Why it matters:** A fake OpenSSH-8.9 honeypot logged 317 MB of traffic over 54 days — first hit in under a minute, ~4,987 connections/day from 7,556 unique IPs. 52% cloud/VPS vs 48% residential, with cloud generating 59% of login volume; one Belgian residential IP alone made 156k attempts. Concrete, current numbers on internet "background radiation" — useful baseline for anyone reasoning about exposed infrastructure (ad servers, RTB endpoints, internal admin ports).
   **URL:** https://arman-bd.hashnode.dev/i-left-port-22-open-on-the-internet-for-54-days-here-s-who-showed-up
   **Discussion:** https://lobste.rs/s/cpu221/i_left_port_22_open_on_internet_for_54_days
