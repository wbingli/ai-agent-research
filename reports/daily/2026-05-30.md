# Tech Digest — 2026-05-30

**1. OpenRouter raises $113M Series B**
OpenRouter’s funding underscores how quickly model-routing and multi-provider inference layers are becoming core AI infrastructure, not just developer convenience. For engineering leaders, the takeaway is that abstraction, reliability, and cost optimization across many LLM providers are now strategic platform concerns. [Read](https://openrouter.ai/announcements/series-b)

**2. Rotary GPU: running large MoE models under limited VRAM**
This paper explores execution paths for large mixture-of-experts models on constrained local GPUs, attacking one of the practical blockers to cheaper, more private inference. It is relevant for teams evaluating whether more workloads can move from hosted APIs to edge, desktop, or owned hardware. [Read](https://arxiv.org/abs/2605.29135)

**3. CVE-2026-48710: a maintainer’s perspective**
A Python maintainer walks through the reality of handling a security CVE from the project side: triage pressure, disclosure mechanics, user expectations, and ecosystem coordination. It is a useful reminder that dependency security is as much an operational process as it is a code fix. [Read](https://marcelotryle.com/blog/2026/05/28/cve-2026-48710-a-maintainers-perspective/)

**4. AV2 video standard v1.0 is released**
The Alliance for Open Media published the final AV2 specification, marking the next major open video codec milestone after AV1. Anyone responsible for media delivery, streaming cost, browser/device compatibility, or long-term video architecture should start tracking encode/decode support timelines. [Read](https://av2.aomedia.org/)

**5. Zig build system reworked**
Zig’s latest devlog details a substantial rework of its build system, part of the language’s broader push toward simpler cross-compilation and reproducible native builds. Even if Zig is not in production for you today, its tooling ideas are worth watching because build complexity remains a major tax on backend and infrastructure teams. [Read](https://ziglang.org/devlog/2026/#2026-05-26)
