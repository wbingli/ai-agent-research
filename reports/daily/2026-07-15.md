# Tech Digest — 2026-07-15

**1. Inkling: Our Open-Weights Model**
Thinking Machines released Inkling, a multimodal MoE model with open weights, controllable reasoning effort, and fine-tuning support via Tinker. For engineering leaders, it is another signal that serious frontier-style capabilities are moving into customizable, self-hostable workflows rather than only closed APIs. [Read](https://thinkingmachines.ai/news/introducing-inkling/)

**2. Grok Build Is Open Source**
xAI open-sourced Grok Build, its coding-agent harness and terminal UI for fullscreen, mouse-interactive agent work. The interesting part is less the brand than the pattern: agent infrastructure is becoming productized, inspectable, and extensible, which should raise expectations for internal developer-platform tooling. [Read](https://github.com/xai-org/grok-build)

**3. Running Gemma 4 26B at 5 tokens/sec on a 13-year-old Xeon with no GPU**
A practical writeup shows a 26B model running on old CPU-only hardware, emphasizing quantization, memory layout, and realistic throughput. It is a useful reminder that inference cost is not just about buying GPUs; software choices can unlock good-enough local AI for constrained or private environments. [Read](https://www.neomindlabs.com/2026/06/08/running-gemma-4-26b-at-5-tokens-sec-on-a-13-year-old-xeon-with-no-gpu/)

**4. Mysteries of Telegram Data Centers**
This deep dive maps Telegram’s account-to-data-center behavior and the practical consequences of DC placement, migration, and routing. For leaders running global systems, it is a concrete case study in how infrastructure topology leaks into user experience, reliability, and operational constraints. [Read](https://dev.moe/en/3025)

**5. SQLite Should Have Rust-Style Editions**
The essay argues SQLite could evolve more safely by adopting opt-in “editions,” similar to Rust, allowing behavior changes without breaking existing applications. It is a sharp framing for platform teams: long-lived APIs need explicit compatibility strategies, not just careful release notes. [Read](https://mort.coffee/home/sqlite-editions/)
