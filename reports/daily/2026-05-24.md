# Tech Digest — 2026-05-24

**1. Constraint Decay: The Fragility of LLM Agents in Backend Code Generation**
A new paper argues that coding agents can produce working backend code while silently drifting away from architectural constraints like ORM patterns, database boundaries, and project structure. For engineering leaders, it’s a reminder that agent adoption needs constraint tests and design review, not just functional test pass rates. [Read](https://arxiv.org/abs/2605.06445)

**2. Cheap code means formal verification is reasonable now**
Antfly shows how TLA+ plus coding agents can make formal models of distributed systems cheaper to create and iterate, turning some race-condition hunts into executable specifications. This matters because AI lowers the cost of verification work that used to be too expensive for many product teams. [Read](https://antfly.io/blog/agent-formal-verification)

**3. Memory has grown to nearly two-thirds of AI chip component costs**
Epoch AI estimates high-bandwidth memory now represents about 63% of AI chip component cost, up sharply as accelerators become more memory-bound. The strategic takeaway: AI infrastructure economics are increasingly about memory supply, bandwidth, and packaging—not just GPU compute. [Read](https://epoch.ai/data-insights/ai-chip-component-cost-shares)

**4. A Network Allow-List Won’t Stop Exfiltration**
This security writeup explains why sandboxed or agent-generated code can still leak secrets through any permitted domain, even with a strict network allow-list. The proposed fix is an L7 egress proxy with data-loss prevention, a pattern worth considering for teams running untrusted AI tooling or plugins. [Read](https://www.dergraf.org/notes/canister-egress-proxy-dlp/)

**5. Reasonix — DeepSeek-native AI coding agent for your terminal**
Reasonix is an open-source terminal coding agent designed around DeepSeek’s prefix-cache economics, with MCP support and planning workflows. It’s notable less as “another agent” and more as a sign that model-specific caching and cost architecture are becoming first-class product differentiators. [Read](https://esengine.github.io/DeepSeek-Reasonix/)
