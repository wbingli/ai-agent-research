# Tech Digest — 2026-06-24

**1. OpenAI unveils its first custom chip**
OpenAI’s Broadcom-built “Jalapeño” inference processor shows the biggest model companies moving deeper into full-stack hardware control to reduce cost, latency, and supplier risk. For engineering leaders, custom silicon is becoming a strategic capacity-planning lever, not just a cloud procurement detail. [Read](https://techcrunch.com/2026/06/24/openai-unveils-its-first-custom-chip-built-by-broadcom/)

**2. Computer use in Gemini 3.5 Flash**
Google introduced built-in computer-use capabilities for Gemini 3.5 Flash, letting the model operate browsers and apps as part of agent workflows. This matters because reliable UI automation is becoming a key interface layer between LLMs and legacy enterprise systems that lack clean APIs. [Read](https://blog.google/innovation-and-ai/models-and-research/gemini-models/introducing-computer-use-gemini-3-5-flash/)

**3. Keeping the Web open and private in the bot era**
Mozilla argues that the flood of bots and AI agents is pushing the web toward more CAPTCHAs, logins, and gatekeeping, and proposes privacy-preserving ways to distinguish legitimate users from abuse. The leadership takeaway: anti-bot architecture is now a product, privacy, and platform strategy problem at once. [Read](https://blog.mozilla.org/en/privacy-security/keeping-the-web-open-and-private-in-the-bot-era/)

**4. NVIDIA’s 45°C cooling design for AI factories**
NVIDIA says its latest AI servers can run with much warmer liquid coolant, cutting data-center water use toward zero while improving the economics of dense GPU clusters. As AI workloads scale, thermal design and facilities engineering are becoming first-order constraints on model-serving capacity. [Read](https://blogs.nvidia.com/blog/liquid-cooling-ai-factories/)

**5. Using the Gini coefficient to plan edge capacity**
Fastly describes using the Gini coefficient to model uneven traffic distribution across its edge network and plan capacity more intelligently. It’s a useful example of borrowing simple statistical tools to improve distributed-systems operations, cache efficiency, and infrastructure spend. [Read](https://www.fastly.com/blog/using-gini-coefficient-plan-edge-capacity)
