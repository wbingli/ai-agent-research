# AI Agent Research Daily Digest — 2026-03-26

## 🔥 Top Discoveries

### 1. Hermes Agent + Self-Evolution (NousResearch)
- **Links:** [hermes-agent](https://github.com/NousResearch/hermes-agent) | [self-evolution](https://github.com/NousResearch/hermes-agent-self-evolution) | [Website](https://hermes-agent.nousresearch.com/)
- **What:** An autonomous agent that "grows with you" — lives on your server, maintains persistent memory, and improves over time. The self-evolution companion uses **DSPy + GEPA (Genetic-Pareto Prompt Evolution)** to automatically evolve skills, prompts, and code.
- **Key features:** Plugin architecture (drop Python files into ~/.hermes/plugins/), native Anthropic provider with Claude Code credential auto-discovery, smart approval system that learns safe commands, OpenClaw migration support.
- **Why interesting:** The self-evolution angle is genuinely novel — using genetic algorithms to optimize agent behavior without GPU training. The agent literally gets better at its job through reflective iteration. This is a step toward agents that don't just execute but adapt.

### 2. DeerFlow 2.0 (ByteDance)
- **Link:** [github.com/bytedance/deer-flow](https://github.com/bytedance/deer-flow)
- **What:** Open-source "SuperAgent" harness — a ground-up rewrite from v1. Orchestrates sub-agents, sandboxes, memory, tools, and skills to handle complex tasks spanning minutes to hours.
- **How it works:** Spawns parallel sub-agents for different task components (scraping, image gen, coding), then converges results. Evolved from a deep research tool into a general-purpose agent orchestrator.
- **Why interesting:** ByteDance open-sourcing their internal agent infrastructure is significant. The sub-agent orchestration pattern with sandboxing and memory is becoming the standard architecture for serious agent systems.

### 3. The Agency (msitarzewski/agency-agents)
- **Link:** [github.com/msitarzewski/agency-agents](https://github.com/msitarzewski/agency-agents)
- **Stars:** 10K+ in 7 days
- **What:** 147 pre-built AI agent role profiles across 12 divisions (engineering, design, growth marketing, product, etc.). Drop-in compatible with Claude Code, Cursor, Aider, Windsurf, Gemini CLI, and GitHub Copilot.
- **How it works:** Plain markdown agent definitions — no runtime overhead, just context. Each agent has a specialized role, responsibilities, and workflows. Started as a Reddit thread.
- **Why interesting:** Not a framework but a pattern — the idea that well-crafted role definitions are the primitive for multi-agent systems. The viral growth (10K stars in 7 days) shows demand for structured agent specialization.

### 4. VoltAgent/awesome-agent-skills
- **Link:** [github.com/VoltAgent/awesome-agent-skills](https://github.com/VoltAgent/awesome-agent-skills)
- **What:** Curated collection of real-world agent skills from official dev teams — compatible with Claude Code, Codex, Antigravity, Gemini CLI, Cursor, and others.
- **Why interesting:** Unlike bulk-generated skill repos, focuses on battle-tested skills from actual engineering teams. The emerging "agent skills" ecosystem (SKILL.md files, skill marketplaces like ClaHub/SkillFlow) is becoming a real thing.

### 5. TurboQuant (Google Research)
- **Link:** [research.google/blog/turboquant](https://research.google/blog/turboquant-redefining-ai-efficiency-with-extreme-compression/)
- **What:** Extreme AI model compression technique. 523 points on HN.
- **Why interesting:** Directly enables running larger models locally — relevant for local agent deployments and edge AI.
