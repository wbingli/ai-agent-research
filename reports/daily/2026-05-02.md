### 4. Copy Fail: 732 Bytes to Root on Every Major Linux Distribution

- **Why it matters:** A small Linux kernel exploit path reportedly chains AF_ALG and `splice()` into a page-cache write primitive to get root across major distributions. Even if the details require careful validation, this is exactly the kind of low-level vulnerability backend and infra teams need on their radar because it affects fleet patching, container-host assumptions, and kernel exposure.
- **Source:** TLDR Tech
- **URL:** https://xint.io/blog/copy-fail-linux-distributions
- **TLDR issue:** https://tldr.tech/tech/2026-05-01

### 5. Mozilla pushes back against Google’s browser Prompt API

- **Why it matters:** The proposed built-in browser Prompt API could make local/browser-provided AI capabilities a web platform primitive, but Mozilla argues it risks centralizing control and weakening openness. This is a strategically important AI platform story: if browsers expose model APIs, web apps may gain powerful new capabilities, but portability, model choice, privacy guarantees, and standards governance become critical.
