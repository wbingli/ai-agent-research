# Tech Digest — 2026-05-31

**1. Cloudflare Turnstile requiring fingerprintable WebGL**
Cloudflare’s bot check appears to require WebGL signals that can be used for browser fingerprinting, highlighting the privacy cost of “invisible” abuse prevention. Engineering leaders should treat bot mitigation as part of the product threat model, not just a security vendor toggle. [Read](https://hacktivis.me/articles/cloudflare-turnstile-webgl-fingerprinting)

**2. Dav2d**
The VideoLAN ecosystem is introducing dav2d, a new AV1 decoder effort aimed at faster, more maintainable media playback. For teams serving or processing video, decoder performance and portability directly affect compute cost, battery life, and user experience. [Read](https://jbkempf.com/blog/2026/dav2d/)

**3. 1-Bit Bonsai Image 4B Image Generation for Local Devices**
PrismML presents 1-bit and ternary Bonsai Image 4B models for image generation on local devices, continuing the push toward smaller, cheaper generative models. If quality holds up, this kind of quantization shifts more AI UX from cloud inference budgets to private, low-latency edge execution. [Read](https://prismml.com/news/bonsai-image-4b)

**4. The radical network redesign that led AWS to forge a more resilient cloud**
AWS describes how random graph theory influenced a data-center network design intended to improve resilience and scaling behavior. It is a useful reminder that hyperscale reliability often comes from changing topology and failure domains, not merely adding more capacity. [Read](https://www.aboutamazon.com/stories/aws-random-graph-theory-data-center-network-design)

**5. Backpressure is all you need**
This essay argues that backpressure is the core control mechanism behind reliable queues, APIs, and distributed systems under load. The operational takeaway is simple: systems that cannot slow producers or shed work deliberately will eventually fail in surprising ways. [Read](https://www.lucasfcosta.com/blog/backpressure-is-all-you-need)
