### 5. NVIDIA Greenboost — Transparently Extend GPU VRAM
- **Link:** https://gitlab.com/IsolatedOctopi/nvidia_greenboost
- **Stars:** 412 points on HN (trending #12)
- **What:** A tool that transparently extends GPU VRAM using system RAM or NVMe storage. Lets you run larger AI models on GPUs with limited VRAM.
- **Why it matters:** For anyone running local LLMs, this is huge. Instead of being constrained by your GPU's VRAM (e.g., 8GB on a 4070), you can spill into system RAM/NVMe. Not as fast as native VRAM, but enables running models that wouldn't fit otherwise.
- **Take:** Practical tool for local AI inference. If you run Ollama or similar, this could unlock bigger models.

## 📡 Ecosystem Updates

### MCP 2026 Roadmap Published
- **Link:** https://blog.modelcontextprotocol.io/posts/2026-mcp-roadmap/
- The Model Context Protocol published its 2026 roadmap (March 9). Four focus areas:
  1. **Streamable HTTP** — Run MCP servers as remote services (not just local stdio)
  2. **Tasks primitive** — Better lifecycle management for long-running agent tasks
  3. **Enterprise readiness** — Audit trails, SSO, compliance
  4. **Standard metadata format** — Registry discovery without live connections
- The shift from local-only to remote MCP servers is the big unlock for production deployments.
