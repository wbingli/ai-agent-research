# Tech Digest — 2026-05-18

**1. Project Glasswing: what Mythos showed us**
Cloudflare tested security-focused frontier models against live infrastructure code and found useful signals, but also clear limits around reliability, context, and operational trust. For engineering leaders, this is a practical look at where AI security review can augment teams today without becoming an unsupervised control plane. [Read](https://blog.cloudflare.com/cyber-frontier-models/)

**2. Cutting inference cold starts by 40x with LP, FUSE, C/R, and cuda-checkpoint**
Modal details how it reduced serverless GPU inference startup latency using lazy loading, FUSE, checkpoint/restore, and CUDA checkpointing. The takeaway is that AI infra performance increasingly depends on deep systems work, not just model serving frameworks. [Read](https://modal.com/blog/truly-serverless-gpus)

**3. Anthropic acquires Stainless**
Anthropic is acquiring Stainless, a company focused on API SDK generation and developer experience. This matters because model companies are moving beyond chat surfaces toward stronger platform ergonomics, typed APIs, and agent-friendly integration layers. [Read](https://www.anthropic.com/news/anthropic-acquires-stainless)

**4. A Linux-like kernel in a browser tab**
Leaning Technologies explains BrowserPod, a WebAssembly architecture that can run many Linux applications concurrently inside the browser. It is a useful glimpse of where local-first compute, sandboxing, and web-delivered developer environments may converge. [Read](https://labs.leaningtech.com/blog/browserpod-deep-dive.html)

**5. Agent Hooks: Deterministic Control for Agent Workflows**
This piece argues for adding deterministic hooks around agent workflows so teams can constrain behavior, inject policy, and make automation auditable. For production AI systems, the important shift is from “let the agent decide” to explicit control points engineers can test and govern. [Read](https://nader.substack.com/p/agent-hooks-deterministic-control)
