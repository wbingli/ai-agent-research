# Tech Digest — 2026-06-22

**1. GLM-5.2 — How to Run Locally**
Unsloth published a practical guide for running Z.ai’s new GLM-5.2 model on local hardware, which is useful as more teams evaluate when self-hosted inference beats hosted APIs on privacy, latency, and cost. [Read](https://unsloth.ai/docs/models/glm-5.2)

**2. Prompt Injection as Role Confusion**
An ICML 2026 project frames prompt injection as a failure in how LLMs represent roles, giving teams a clearer mental model for why attacks work and when they are likely to succeed. This is directly relevant to anyone shipping agents that mix user instructions, tool outputs, and system policy. [Read](https://role-confusion.github.io)

**3. Deno Desktop**
Deno is adding a path for building self-contained desktop apps with framework detection, hot reload, native windows, auto-update, and cross-platform distribution. It is another sign that lightweight web-native runtimes are pushing into territory historically owned by Electron-style stacks. [Read](https://docs.deno.com/runtime/desktop/)

**4. Agent Hooks for Deterministic Guardrails**
This writeup argues that agent safety and coding standards should be enforced with deterministic hooks rather than hoping prompts are followed. For engineering leaders adopting AI coding agents, the key takeaway is to move policy into tooling: tests, linters, pre-commit checks, and explicit hook points. [Read](https://zarar.dev/agent-hooks-deterministic-guardrails-for-ai-generated-code/)

**5. Moebius: 0.2B Image Inpainting with 10B-Level Performance**
Moebius is a lightweight image-inpainting model claiming 10B-class performance with only 0.2B parameters. If the results hold up, it points toward cheaper media-generation features that can run closer to users or inside constrained production pipelines. [Read](https://hustvl.github.io/Moebius/)
