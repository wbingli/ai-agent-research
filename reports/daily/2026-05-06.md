# Tech Digest — 2026-05-06

**1. Agents can now create accounts, buy domains, and deploy apps**
Cloudflare is showing agents that can complete real operational workflows end-to-end: provisioning accounts, paying with Stripe, registering domains, and deploying projects. The core shift is agents moving from “assistant in a chat box” to controlled operators that can touch billing, identity, DNS, and production infrastructure.

**2. Faster Gemma inference with multi-token prediction**
Google is using small draft models to predict multiple future tokens, then letting the main model verify them, which can speed up generation without changing model quality much. This matters because inference cost is increasingly about serving efficiency, not just model size.

**3. Computer-use agents are far more expensive than structured APIs**
A writeup compares browser/computer-control agents against direct API integrations and finds the UI-driving approach can be dramatically more expensive. Good reminder: use agents for messy, unstructured workflows, but expose clean APIs/tools whenever you control the system.

**4. Nix/Lix local privilege escalation**
The Nix ecosystem has a local privilege-escalation issue tied to Lix/Nix behavior, with a separate integer-overflow writeup explaining the exploit path. Worth watching if you use Nix on developer machines or CI hosts, because build tooling often has broader trust than people realize.

**5. Formatting 25 million lines overnight**
Stripe’s rubyfmt story is about rolling out an automated formatter across a massive codebase without grinding development to a halt. The interesting lesson is not formatting itself; it is how to land huge mechanical changes safely through tooling, batching, ownership, and minimizing merge pain.
