## 🔥 Top Discoveries

### 1. Cook — CLI Orchestrator for Coding Agents
**Link:** https://github.com/rjcorwin/cook | [HN: 296 pts](https://news.ycombinator.com/front)
**What:** A composable CLI that orchestrates Claude Code, Codex, and OpenCode with simple primitives. You can run review loops (`cook "task" review`), race multiple attempts (`cook "task" v3 "best"`), compare approaches (`cook "A" vs "B" pick "criteria"`), and chain operators left-to-right.
**Why interesting:** Elegant abstraction over coding agents — treats them as composable unix-style primitives. The `ralph` (review-all-loop-pick-halt) pattern is clever for autonomous multi-pass coding. 296 HN points shows real developer interest.

### 2. Hindsight — Agent Memory That Learns (vectorize-io)
**Link:** https://github.com/vectorize-io/hindsight
**Stars:** Trending on GitHub (week of Mar 15)
**What:** Agent memory system focused on *learning* not just *remembering*. Claims SOTA on LongMemEval benchmark, outperforming RAG and knowledge graph approaches. Drop-in LLM wrapper — add memory with 2 lines of code. Independently benchmarked by Virginia Tech and Washington Post.
**Why interesting:** Memory is the biggest gap in current agent systems. Most solutions just do vector search over conversation history. Hindsight claims to actually learn patterns from interactions. Already in production at Fortune 500s. Has a Claude Code skill for instant docs access.

### 3. Kitten TTS — Sub-25MB Text-to-Speech Models
**Link:** https://github.com/kittenml | [HN: 486 pts](https://news.ycombinator.com/item?id=47441546)
**What:** Three new TTS models: 80M, 40M, and 14M params. The smallest is under 25MB and runs on CPU. Claims SOTA expressivity among similar-sized models. Python API: `KittenTTS("KittenML/kitten-tts-micro-0.8")`.
**Why interesting:** TTS small enough to run on embedded devices, edge, or as a sidecar in any agent system. 486 HN points — massive interest. Could enable voice for IoT/robotics agents without cloud dependency.

### 4. Xiaomi's "Hunter Alpha" — Mystery Stealth Model on OpenRouter
**Link:** [Reuters](https://www.reuters.com/business/media-telecom/mystery-ai-model-has-developers-buzzing-is-this-deepseeks-latest-blockbuster-2026-03-18/)
**What:** On March 11, two anonymous models ("Hunter Alpha" and "Healer Alpha") appeared on OpenRouter with no attribution. Developers speculated it was DeepSeek V4 in stealth testing. Turned out to be Xiaomi. The model apparently performed competitively enough to fool the community.
**Why interesting:** Xiaomi entering the frontier model space is significant. Also shows OpenRouter becoming the de facto testing ground for new models, and that Chinese tech cos beyond DeepSeek are shipping competitive LLMs.

### 5. Dimensional OS (dimos) — Agentic OS for Robotics
**Link:** https://github.com/dimensionalOS/dimos
**What:** "Agentic operating system for physical space" — program humanoids, quadrupeds, drones in natural language. Integrates cameras, lidar, actuators. Multi-agent systems for hardware. MCP support. Pre-release beta.
**Why interesting:** Bridges the gap between software agents and physical robotics. Natural language → robot control with MCP integration means existing agent tooling can extend to physical world. Early but ambitious.
