# Tech Digest — 2026-05-20

**1. An OpenAI model has disproved a central conjecture in discrete geometry**
OpenAI says one of its models solved the 80-year-old unit distance problem, a notable example of AI producing new mathematical knowledge rather than just assisting with proof search. For engineering leaders, it is another signal that frontier models are moving from coding copilots toward research systems that can expand design spaces. [Read](https://openai.com/index/model-disproves-discrete-geometry-conjecture/)

**2. GitHub confirms breach of 3,800 repos via malicious VS Code extension**
A GitHub employee installed a malicious VS Code extension that exposed roughly 3,800 internal repositories, showing how developer tooling and editor supply chains remain high-value attack paths. Teams should treat extensions like production dependencies: inventory them, restrict publisher trust, and monitor for credential or source exfiltration. [Read](https://www.bleepingcomputer.com/news/security/github-confirms-breach-of-3-800-repos-via-malicious-vscode-extension/)

**3. Incident Report: GCP account suspension took Railway offline**
Railway’s postmortem says Google Cloud incorrectly suspended its account, causing a platform-wide outage for GCP-hosted infrastructure. The incident is a useful reminder that cloud-provider control-plane and billing/account systems are part of your reliability architecture, even when your own data plane is healthy. [Read](https://blog.railway.com/p/incident-report-may-19-2026-gcp-account-outage)

**4. Gemini 3.5 Flash: more expensive, but Google plans to use it for everything**
Simon Willison breaks down Google’s new Gemini 3.5 Flash release, including pricing and the apparent strategy of using the model broadly across Gemini, Search, Antigravity, and APIs. The takeaway for AI product teams is that “fast” models are becoming default platform primitives, not just cheap fallback tiers. [Read](https://simonwillison.net/2026/May/19/gemini-35-flash/)

**5. Logic bug in Linux kernel __ptrace_may_access() disclosed as CVE-2026-46333**
Qualys published a technical advisory for a Linux kernel access-control bug in `__ptrace_may_access()`, with case studies covering common privileged helpers. Security and platform teams should track kernel fixes quickly because ptrace permission bugs can undermine container, debugging, and privilege-boundary assumptions. [Read](https://cdn2.qualys.com/advisory/2026/05/20/cve-2026-46333-ptrace.txt)
