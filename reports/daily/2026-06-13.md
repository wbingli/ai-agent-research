# Tech Digest — 2026-06-13

**1. 21 Zero-Days in FFmpeg**
An autonomous security agent found 21 FFmpeg vulnerabilities, including issues latent for 15–20 years, and produced reproducible PoC inputs plus an RCE primitive. For engineering leaders, this is a sharp reminder that media parsers remain high-risk infrastructure and that AI-assisted security work is moving from demos to actionable findings. [Read](https://depthfirst.com/research/21-zero-days-in-ffmpeg)

**2. Pyodide 314.0: WebAssembly Wheels for PyPI**
Pyodide 314.0 lands after PEP 783 acceptance, letting Python packages publish Emscripten/WebAssembly wheels directly to PyPI. This could significantly reduce friction for Python-in-browser apps, notebooks, edge sandboxes, and product surfaces that need Python without server-side execution. [Read](https://blog.pyodide.org/posts/314-release/)

**3. Homebrew 6.0.0**
Homebrew’s new major release adds tap trust, a faster and smaller default internal JSON API, Linux sandboxing, bundle improvements, and early macOS 27 support. It matters because package managers sit in the software supply chain’s critical path; better trust and sandboxing defaults reduce operational risk for developer fleets. [Read](https://brew.sh/2026/06/11/homebrew-6.0.0/)

**4. Software Is Made Between Commits**
Zed introduces DeltaDB, a version-control layer that records fine-grained editing operations and links them to the conversations with humans or agents that produced them. If agentic coding becomes normal, teams will need auditability and collaboration primitives below the commit level, not just pull-request snapshots. [Read](https://zed.dev/blog/introducing-deltadb)

**5. AI Agent Bankrupted Its Operator While Trying to Scan DN42**
A cautionary post describes an AI agent attempting to scan the DN42 network and running up a reported $6,531 AWS bill. The lesson is operational, not just funny: autonomous agents need hard budgets, network egress controls, scope limits, and kill switches before they touch cloud infrastructure. [Read](https://lantian.pub/en/article/fun/ai-agent-bankrupted-their-operator-scan-dn42lantian.lantian/)
